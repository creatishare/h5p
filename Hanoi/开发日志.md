## 玩法（v3：极简，对齐教学）

- **目标**：把所有圆盘从 A 柱搬到 C 柱
- **规则**：每次只搬 1 个；只能搬最上面那个；大盘不能放小盘上
- **唯一模式：手动**
  - 点击柱子 → 选中顶部圆盘 → 再点目标柱放置
  - 或：按住顶部圆盘拖动到目标柱（拖动阈值 5px，小于此距离仍按点击处理）
- **圆盘数量**：2 / 3 / 4 三档，默认 3
- **重置**：把当前关卡退回到全部在 A 的初始状态（不会改 N）
- **当前步数 / 最少步数**：放在左侧"操作记录"栏顶部，stat 卡形式呈现
- **目标和规则**：放在左侧栏最顶部的卡片中（蓝色渐变背景），便于学生随时回看

## ⛔ 红线：不要动的约束（与 climb-s 同源）

### 1. 客户端桥接（`Hanoi.html` 的 `<head>` 内联 + `app.js` 顶部兜底）

`__setIframeUrlHandled` 守卫避免 native → SetIframeUrl → 我们 SendMessageToU3d
→ native 又触发 page-loaded 的反馈环。

主战场在 `<head>` 内联，必须比客户端注入的桥接调用更早就位：

```html
<script>
  window.SendMessageToJs = function () {};
  var __setIframeUrlHandled = false;
  window.SetIframeUrl = function (url) {
    if (__setIframeUrlHandled) return;
    __setIframeUrlHandled = true;
    window.SendMessageToU3d && window.SendMessageToU3d("PageLoaded");
    window.OnIframeLoad && window.OnIframeLoad();
    if (url && url !== location.href) {
      location.href = url;
    }
  };
  window.ShowClose = function () {};
  window.HideClose = function () {};
</script>
```

### 2. 文件名 / 路径只能用 ASCII

不要起任何中文 / 全角字符的文件名（图片、样式都不行）。客户端 WebView 与
线上服务器的 URL 编解码不对称会让 `<img src>` 在第二次设置时失效。

### 3. 单一的"锁定输入"机制

`body.is-animating` 在单次点击-放置飞行动画期间（~460ms）锁住所有交互
（peg click / 重置 / 圆盘下拉 / 顶栏 tab）。源头守卫：`handlePegClick` /
`tryMove` / `activateDrag` 都先检查 `state._animating`，确保 CSS 拦截
被绕过时逻辑层也不会推进状态。

v2 曾经引入过 `body.is-auto-running` 用于自动模式的长任务锁，
v3 移除自动模式之后这个机制不再需要。

### 4. 动画 token

每次新动画前 `state._animationToken++`。旧 token 的 `.then(...)` 回调
发现 `token !== state._animationToken` 就静默退出。`initGame()`（重置
/ 改 N）会自动让旧 token 失效。

### 5. 通知客户端 `PageLoaded`

`init()` 末尾调用 `SendMessageToU3d("PageLoaded")`。客户端用这个信号决定何
时关掉 loading 遮罩。

### 6. Web Audio 合成音效，不引入外部 mp3

- `move`：每一步搬运的"抬起 + 落下"两个三角波音符
- `complete` / `finish`：通关时上扬的三 / 四和弦
- `error`：低频方波（不能放大盘到小盘上的反馈）
- `toggle`：开关声音的提示音

### 7. 拖动的两阶段激活（与原版一致，沿用到此版）

`pointerdown` 只暂存 `pendingDrag = {peg, diskEl, startX, startY}`，不立即
进入拖动态。`pointermove` 超过 5px 阈值才 `activateDrag()`，否则在
`pointerup` 时清空 `pendingDrag` 让 click 事件正常落到 peg-zone 上。

这样：**轻点 = 选中**，**拖动 = 拖动**，互不干扰。

## 与 climb-s 的差异一览

| 维度 | climb-s（爬楼梯） | Hanoi（汉诺塔） |
|---|---|---|
